## white-board-collaborator

## Installation

```bash
npm i
```
# Setup Environment Variables

Create a .env file with the following variables:

```bash
CONVEX_DEPLOYMENT=
NEXT_PUBLIC_CONVEX_URL=
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
LIVEBLOCKS_SECRET_KEY=
```
# Setup Convex
```bash
npx convex dev
```
# Start the Application
```bash
npm run dev
```
